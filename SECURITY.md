# Security Policy

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report security issues to **opensource@amadeus.com** and include:

- Affected repository or workflow
- Steps to reproduce
- Impact and scope
- Any proof of concept or logs

We will review reports promptly and coordinate remediation as needed.

If the issue belongs to the upstream Prompt Registry CLI project, please also
report it there so the core maintainers can address it directly.
